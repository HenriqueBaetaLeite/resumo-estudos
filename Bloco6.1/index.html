Formulários HTML

Uso das tags: input, button, form, textarea, select, entres outras.

Para a tag input, existe o atributo type, que contém vários possibilidades, desde o texto até número, assim como cor.

Além do atributo type para o input, existem outras centenas de propriedades que podem ser usadas de acordo com a necessidade.

Uso a propriedade for do label em conjunto com o id do input, para dizer que a label pertence àquele input.

Colocando a tag input (type radio ou checkbox) dentro da tag label, o "nome/descrição" passa a ser clicável.

Caso use submissão de arquivos no formulário, utilizar a propriedade enctype com o valor multipart/form-data.
