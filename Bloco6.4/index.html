CSS Responsivo

Construir páginas responsivas é uma habilidade fundamental para pessoas que trabalham com desenvolvimento front-end hoje em dia. Quando falamos sobre páginas responsivas, o termo " mobile first " - layout voltado para dispositivos móveis - vem à cabeça imediatamente. Em teoria, é mais fácil construir um layout "mobile first" quando você está começando o desenvolvimento, pois você não tem que se preocupar muito sobre funcionalidades complexas que podem entrar em telas grandes de desktop .

Podemos criar dois arquivos CSS, um para mobile e outro para desktop, talvez um outro ainda para telas muito grandes.
