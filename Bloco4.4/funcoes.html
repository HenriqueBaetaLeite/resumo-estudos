JavaScript - Funções

Funções são a forma mais essencial de organização de código e de lógica na programação. Através de funções, você consegue empacotar partes do seu código e separá-lo em blocos lógicos autocontidos.

Citando Leonardo Leitão do COD3R, "Se o JavaScript fosse um país, objetos seriam o presidente e funções o vice-presidente".

Costumo fazer uma analogia com o controle remoto da televisão. Cada botão tem uma funcionalidade, que executa determinada ação, ou seja, pra cada botão existe um função que foi programada para executar uma tarefa. O botão de ligar/desligar, por exemplo, verifica o estado atual da TV, se estiver desligada, a função liga a TV e vice-versa.

Sintaxe da função:

function nomeDaFuncao(param1, param2) { // bloco de código return param1 + param2 }
