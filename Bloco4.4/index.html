JavaScript - Objetos / For in

Objetos

Objetos são agrupamentos, containers para variáveis, dados.

const meuObjeto = { name: 'Henrique', age: 37, married: true, children: { son1: 'João', son2: 'Ana', }, lastName: 'Leite', };

Com objetos acessamos os valores através de suas chaves, usando a notação de . (ponto) ou também usando [] (colchetes).

console.log(meuObjeto.name); // 'Henrique'

console.log(meuObjeto['married']); // true

Para criar uma nova chave, meuObjeto.nickName = 'Baêta';

console.log(meuObjeto.nickName); // 'Baêta'

console.log(`O nome completo é ${meuObjeto.name} ${meuObjeto.lastName}.`) // O nome completo é Henrique Leite.

For in

O loop for in facilita percorrer um objeto.

for (let key in meuObjeto) {console.log(key, meuObjeto[key])}

// name Henrique

// age 37

// married true

// children { son1: 'João', son2: 'Ana' }

// lastName Leite
