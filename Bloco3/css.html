Parte I

Criando primeiras regras com CSS

Os estilos servem para dar "vida" ao HTML. Podemos colocar cores, animações e outras cositas mas.

Podemos setar esse estilos diretamente na tag, como fiz acima, ou temos a opção de criar uma tag < style > dentro do head ou também um arquivo externo .css

Usando seletores, podemos colocar o atributo 'id' numa tag, o valor desse id deve ser único.

na tag style, colocamos o identificador # antes do nome do id desejado (#id-da-tag), abrimos chaves (#id-da-tag { color: blue;}), e lá setamos a propriedade que queremos mudar.

Também podemos usar o atributo 'class' nas tags, o que diferencia é que ele pode ser usado em várias tags. Dentro do style usamos o identificador . (ponto) para 'class'.
