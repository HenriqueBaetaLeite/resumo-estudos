Parte II

Estruta das tags HTML

a tag pode ou não conter um fechamento '< />' ou '<> < /> < br > < /br >'

Uma tag contém o nome, e pode conter atributos/propriedades.

Uma tag é um objeto que possui várias chaves.

A estrutura da página HTML é composta por pela tag < head >, onde contém outras tags como meta e o título da página, onde aparece no browser.

Depois temos a tag < body >, que vai conter o conteúdo da página, o que de fato irá aparecer no browser para o usuário.

Dentro da tag body, temos diversas tags, cada uma com seu significado, por exemplo as tags de títulos, h1, h2 até h6, ou tags de texto/parágrafo, assim como tags que separam conteúdos, como a tag main, section, div, entre outras.

Também temos as tags de mídias, como img, audio, video, canvas, etc.

Algumas tags precisam de atributos para funcionar, como a tag img, que necessita do atributo src, que é o source (fonte) da imagem, podendo ser um caminho interno, ou uma url externa.

Aqui uma tag de imagem sem source, aí o que vai aparecer é o alt, que seria a descrição da imagem, caso ela não carregue

Parte III

Links no HTML

tag < a > serve para redirecionar para outro link. Ela possui o atributo href que é o "endereço" para esse link, e pode conter outros atributos importantes, como o target, onde podemos dizer que o link será aberto em outra aba ou página do browser
