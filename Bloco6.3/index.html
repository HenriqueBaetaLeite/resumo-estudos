CSS Flexbox

Parte I

Flexbox é um módulo de layout que torna fácil o desenvolvimento de estruturas responsivas.

Visa fornecer uma maneira mais eficiente de organizar, alinhar e distribuir o espaços entre elementos dentro um container/box/caixa.

A ideia principal por trás do "layout flexível" é dar ao container a habilidade de alterar o tamanho ( width e height ) e a ordem dos itens para melhor ocupar o espaço disponível. Essa habilidade nos permite acomodar uma página em todos os tipos de dispositivos e tamanhos de tela.

A maneira mais fácil que encontrei para "dominar" o CSS é imaginar os elementos HTML de uma página como caixa, e como você quer organizar para que essas caixas apareçam na tela.

Exercícios do dia neste repositório: https://github.com/tryber/exercise-tech-gallery
