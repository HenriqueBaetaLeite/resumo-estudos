JavaScript - Array

Manipulação de Arrays.

Um array é basicamente um agrupamento de variáveis.

const meuArray = ['ola', 2, true]

Consigo acessar os elementos do array através da sua posição, do seu índice. A primeira posição é o 0.

meuArray[0] // 'ola'

meuArray[2] // true

Existem diversos métodos para manipulação de arrays. Conseguimos inserir, remover, substituir valores, etc.
