Testes com JQuery

Para acessar um elemento basta usar $('.nome-da-class')... Daí podemos escolher o tipo de "manipulação", seja adicionar um texto, adicionar uma classe, id, atributo, entre outras opções.

Alguns exemplos:

  • $('#meu-id')
  • $('.minha-classe')
  • $('p') - Seleciona todas as tags p encontradas
  • $('p.intro') - Seleciona todas as tags p com a classe "intro"
  • $('[href]') - Seleciona todos os elementos que tenham o atributo/propriedade href
  • $('h2[data-testid="title-test"]') - Seleciona todos os h2 que tenham o data-testid igual a title-test
  • $('#myId').attr('width', '500') - Seleciona o elemento com o id myId e seta o width para 500px
  • $(selector).attr({ data-testid: 1, required:true, ... } - Através de um objeto consigo setar vários atributos de vez só)
  • $(selector).attr('id') - Passando apenas 1 argumento, retorna o atributo id do seletor.
  • $(selector).text() .html() .val() - Esses 3 métodos retornam ou setam o texto, conteúdo do elemento HTML e o valor de campos de formulário (inputs).
  • $(selector).text('Hello World') - Passando parâmetro para o método ele seta o que foi passado.
  • Esses métodos recebem como segundo (opcional) parâmetro uma callback, com 2 parâmetros: o index do atual elemento e o original (antigo) valor.
  • Outros 2 métodos bem utilizados são o .append() e o prepend().
  • $(selector).append('Algum texto...') - Eu insiro um texto no final do seletor.
  • $(selector).prepend('outro texto') - Insiro outro texto no início do seletor.
  • Métodos de remoção:
  • $(selector).remove() .empty() .hide() .show() - Remove: remove o elemento e seus filhos; empty: remove os filhos do elemento; hide: esconde os filhos do elemento, show: mostra os filhos do elemento.
  • Métodos para manipulação de CSS:
  • $(selector).addClass() .removeClass() .toggleClass() - adiciona a classe, remove a classe, se tiver a classe remove, se não tiver adiciona.
  • $('#meu-id').css('background-color') - Retorna o valor do background-color do seletor ou o primeiro encontrado.

Existem outras dezenas ou centenas de métodos, para eventos, selecionar parentes, efeitos e até animação, entre outros.
