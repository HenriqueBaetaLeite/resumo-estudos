JavaScript - Lógica de Programação

Lógica de programção consiste numa série de regras e normas baseadas na lógica matemática e em teorias da computação, que tem como objetivo facilitar o raciocínio no momento da criação de uma cadeia de comandos.

A lógica seria a organização das instruções de um programa para que o objetivo seja alcançado, para que o problema seja resolvido.

"O computador é burro, mas extremamente obediente, eficiente e veloz!"

Quanto mais treino, melhor o raciocínio fica.
